## Cellular Automaton

A tape is unidimensional with sequential forward or backward moviments. 

For simplicity each position have 1 bit of information.

A Wolfram Engine define a rule as a transform applied recursivement to a sequence values V(n), based in the values of the values of V(n-1), V(n), V(n+1). 

Using a sequence bits,  :

| pattern | 111 | 110 | 101 | 100| 011 | 010 | 001 | 000 |
| - | - | - | - | - | - | - | - | - |
| new state | x | x | x | x | x | x | x | x |
